Quick Installation Guide

1

Add to your lazy config

Copy the plugin configuration to a new file in ~/.config/nvim/lua/naveen/lazy/

2

Restart Neovim

Exit and reopen Neovim to reload the configuration

3

Install plugins

Run :Lazy sync to install and update all plugins

Productivity Boosters

toggleterm.nvim

Recommended

Persist and toggle multiple terminals during an editing session. Great for running tests and builds.

Key Features

  • Multiple terminal instances
  • Floating & split terminals
  • Send commands to terminal
  • Custom terminal functions
‚ñ∂ View Configuration 
{
    "akinsho/toggleterm.nvim",
    version = "*",
    config = function()
        require("toggleterm").setup({
            size = 20,
            open_mapping = [[]],
            direction = "float",
            float_opts = {
                border = "curved",
            },
        })
        
        -- Custom terminals
        local Terminal = require("toggleterm.terminal").Terminal
        local lazygit = Terminal:new({ cmd = "lazygit", hidden = true })
        
        function _lazygit_toggle()
            lazygit:toggle()
        end
        
        vim.api.nvim_set_keymap("n", "lg", "lua _lazygit_toggle()", {noremap = true, silent = true})
    end
}

rest.nvim

Fast Neovim HTTP client written in Lua. Test APIs directly from your editor.

Key Features

  • Environment variables
  • Response preview
  • cURL command generation
‚ñ∂ View Configuration 
{

        require("rest-nvim").setup({
